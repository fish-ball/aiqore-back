# 前端启动说明

## 快速启动

### 1. 安装依赖

在 `frontend` 目录下运行：

```bash
npm install
```

如果使用 yarn：

```bash
yarn install
```

如果使用 pnpm：

```bash
pnpm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端应用将在 `http://localhost:3000` 启动。

### 3. 确保后端服务运行

前端需要后端 API 服务支持。请确保：

1. 后端 FastAPI 服务正在运行在 `http://localhost:8000`
2. 后端已配置 CORS，允许前端访问

### 4. 访问应用

打开浏览器访问：`http://localhost:3000`

## 常见问题

### 1. 端口被占用

如果 3000 端口被占用，Vite 会自动尝试下一个可用端口，或修改 `vite.config.js` 中的 `server.port` 配置。

### 2. 无法连接后端

- 检查后端服务是否运行
- 检查后端地址是否正确（默认 `http://localhost:8000`）
- 检查浏览器控制台是否有 CORS 错误

### 3. 依赖安装失败

- 清除缓存：`npm cache clean --force`
- 删除 `node_modules` 和 `package-lock.json`，重新安装
- 检查网络连接和代理设置
