# 前端

## 概述

AIQore 前端为基于 Vue 3 + Vite + Element Plus 的个人投资管理系统界面，与后端 FastAPI 通过 REST API 通信。

## 技术栈

- Vue 3.3+ (Composition API)
- Vite 5.0+
- Vue Router 4.2+
- Pinia 2.1+（状态管理）
- Element Plus 2.4+（UI）
- ECharts 5.4+（图表）
- Axios 1.6+（HTTP）

## 功能模块

- **仪表盘**：账户概览、最近交易
- **账户管理**：创建账户、列表、从 QMT 同步
- **持仓管理**：持仓列表、同步、盈亏
- **交易记录**：记录交易、历史查询、分页
- **行情查询**：搜索股票、实时行情、K 线图
- **数据分析**：账户汇总、持仓分析、交易统计、盈亏趋势
- **证券/板块**：证券列表、从 QMT 更新（异步任务）、板块信息
- **数据源配置**（待实现）：数据源连接增删查改、设为行情源/交易源；证券同步任务发起时选择数据源。详见 [8-数据源抽象-前端对接](./8-数据源抽象-前端对接.md)。

## 项目结构

```

│   └── style.css

```

## 快速启动

### 安装依赖

```bash
cd frontend
npm install
```

若 Windows PowerShell 报脚本执行策略错误，可：

- 以管理员打开 PowerShell，执行：`Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`
- 或使用 CMD 在 frontend 下执行 `npm install`

### 开发模式

```bash
npm run dev
```

默认：http://localhost:3000

### 构建与预览

```bash
npm run build    # 产物在 dist/
npm run preview  # 预览生产构建
```

## 配置说明

### API 代理

开发时 Vite 将 `/api` 代理到后端 `http://localhost:8000`。修改后端地址请编辑 `vite.config.js` 的 `server.proxy`。

### 后端依赖

需先启动后端 FastAPI（默认 http://localhost:8000），并确保 CORS 已配置（当前允许所有来源）。若无法请求接口，请检查后端是否运行及控制台是否有 CORS 报错。

## 常见问题

- **端口占用**：Vite 会尝试其他端口，或修改 `vite.config.js` 中 `server.port`。
- **依赖安装失败**：可尝试 `npm cache clean --force`，删除 `node_modules` 与 lockfile 后重新 `npm install`；国内可配置镜像如 `npm config set registry https://registry.npmmirror.com`。
