# 架构与概览

## 项目定位

基于国金 QMT 驱动的个人投资管理系统（AIQore），支持行情浏览、交易管理、持仓跟踪和投资收益分析。

## 功能特性

- **行情浏览**：实时行情、历史 K 线、股票搜索
- **投资账本**：交易记录管理、持仓跟踪、账户管理
- **收益分析**：收益率计算、持仓分析、盈亏统计与趋势
- **QMT 集成**：连接国金 QMT 获取行情与交易数据
- **证券与板块**：证券基础信息同步、板块信息、异步更新任务

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端框架 | FastAPI |
| ORM | SQLAlchemy |
| 数据分析 | Pandas |
| 异步任务 | Celery + Redis |
| 数据源/交易 | 国金 QMT（MiniQMT / xtquant） |
| 前端 | Vue 3 + Vite + Element Plus + Pinia + ECharts |
| 包管理 | 后端 uv / 前端 npm |

## 模块职责简述

| 模块 | 职责 |
|------|------|
| **api/** | HTTP 路由：行情、交易、分析、证券、板块、调试；向 `main.py` 注册。 |
| **models/** | ORM 模型：账户、交易、持仓、证券、板块。 |
| **services/** | 业务逻辑：QMT 连接、行情、交易、分析、证券、板块；被 API 或 Celery 任务调用。 |
| **tasks/** | Celery 任务：如证券信息异步更新；由 `app.celery_app` 的 `include=["app.tasks"]` 加载。 |
| **utils/** | 通用工具：如任务锁等。 |
