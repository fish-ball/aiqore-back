# 开发与运行

## 环境要求

- Python 3.11（见 `.python-version`）
- Node.js（前端开发）
- Redis（Celery 消息队列，可选端口 6379）
- 数据库：PostgreSQL 或 SQLite（见 [3-数据库与迁移](./3-数据库与迁移.md)）

## 包管理：UV

项目使用 [uv](https://github.com/astral-sh/uv) 管理 Python 依赖，替代 pip + requirements.txt。

### 安装 UV

**Windows (PowerShell):**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

**Linux/macOS:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

或：`pip install uv`

### 常用命令

以下命令均在 `backend/` 目录下执行：

```bash
# 同步依赖（创建/更新虚拟环境并安装依赖）
cd backend
uv sync

# 运行后端
uv run python run.py

# 运行 Celery Worker
uv run celery -A app.celery_app worker --loglevel=info --pool=solo

# 运行其他脚本
uv run python script.py

# 添加/移除依赖
uv add package-name
uv remove package-name

# 更新依赖
uv sync --upgrade
```

说明：无需手动激活虚拟环境，`uv run` 会自动使用项目虚拟环境。若需激活：Windows 下 `.venv\Scripts\activate`，Linux/macOS 下 `source .venv/bin/activate`。

### 迁移说明

项目已从 pip + requirements.txt 迁移到 uv + pyproject.toml。依赖定义在 `pyproject.toml`，锁定文件为 `uv.lock`（建议纳入版本控制）。详见历史文档 MIGRATION_TO_UV.md / UV_SETUP.md（内容已整合到本文档）。

## 配置环境变量

复制 `.env.example` 为 `.env` 并填写：

```bash
cp .env.example .env
```

需配置项包括：数据库连接（DB_*）、Redis（REDIS_*）、QMT（XT_QUANT_*）等，见各专题文档。

## 启动方式

### 后端 FastAPI

```bash
cd backend
uv run python run.py
# 或
uv run uvicorn app.main:app --reload
# 或 Windows 下
start_server.bat
```

默认地址：http://localhost:8000  
API 文档：http://localhost:8000/docs 、 http://localhost:8000/redoc

### 前端开发服务器

```bash
cd frontend
npm install
npm run dev
```

默认地址：http://localhost:3000，API 请求通过 Vite 代理到后端（见 [4-前端](./4-前端.md)）。

### Celery Worker（异步任务）

需先启动 Redis（见 [7-异步任务](./7-异步任务.md)），然后在 `backend/` 目录下执行：

```bash
uv run celery -A app.celery_app worker --loglevel=info --pool=solo
# 或 Windows 下
start_celery_worker.bat
```

Windows 上必须使用 `--pool=solo`。

### 数据库初始化

首次或迁移后，在 `backend/` 目录下执行：

```bash
uv run python init_db.py
# 或使用 Alembic（推荐）
uv run alembic upgrade head
```

Windows 下 Alembic 编码问题见 [3-数据库与迁移](./3-数据库与迁移.md)。

## VS Code 任务

`.vscode/tasks.json` 中已配置：

- UV: 启动 FastAPI 服务器
- UV: 启动 Celery Worker
- UV: 初始化数据库 / Alembic 升级 / 创建迁移
- 前端: 启动开发服务器 / 构建生产版本
- 启动完整环境（含 Celery）

可通过「终端 > 运行任务」选择执行。

## 小结

| 目标 | 命令 |
|------|------|
| 后端 | `cd backend && uv run python run.py` 或 `cd backend && start_server.bat` |
| 前端 | `cd frontend && npm run dev` |
| Celery | `cd backend && uv run celery -A app.celery_app worker --loglevel=info --pool=solo` 或 `cd backend && start_celery_worker.bat` |
| 数据库 | `cd backend && uv run python init_db.py` 或 `cd backend && uv run alembic upgrade head` |
